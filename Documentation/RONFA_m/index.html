currentStates

@property (strong,
    nonatomic)NSMutableSet* currentStates;
Discussion

The current states in the construction and running of the automaton. There are several as the running is done in parallel.

finalStates

Contains all the states of the automaton that indicate a pattern match, i.e. those that have finality=YES. 

@property (strong,
    nonatomic)NSMutableSet* finalStates;
Discussion

Note that a set contains strong references and the set must be strong. Hence, retain cycles of states pointing back to the NFA must be avoided. This is no problem as long as the state is automaton-agnostic.

initialState

The starting state of the automaton. 

@property (strong,
    nonatomic)ROState* initialState;
Discussion

Contains pointers to all the other states; hence, it need be the only strong reference to a state.
